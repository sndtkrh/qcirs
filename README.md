# QCirS

## Overview
QCirS is a quantum circuit simulator.

## Requirements
- C++ compiler compatible with C++14 or later
    - g++ 6.4 is recomended.
- Boost C++ Libraries
    - to construct parser using boost::spirit::qi
- CMake 2.8 or later

## Building
```
$ cd <path_to_repository>/
$ mkdir build
$ cd ./build
$ cmake ..
$ make
```
